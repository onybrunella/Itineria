# Scope - MVP vs V2
 
## MVP (à livrer en tout premier)
- Créer un compte / se connecter (auth simple)
- Créer un voyage (titre, dates, description courte)
- Ajouter une étape à un voyage (lieu, coordonnées géo, date, note, une ou plusieurs photos)
- Visualiser la carte du voyage avec les étapes placées et reliées dans l'ordre chronologique
- Visualiser le carnet en scroll (liste des étapes avec photos/notes)
- Modifier / supprimer une étape
## V2 (une fois le MVP fonctionnel et stable)
- Timeline cliquable qui recentre la carte
- Mode "récap" animé en fin de voyage (stats : distance, nombre de lieux, durée)
- Partage public d'un voyage via lien unique
- Détection automatique de l'ordre chronologique et calcul de distance totale
- Mode collaboratif (plusieurs contributeurs sur un même voyage)
## V3 (à reconfirmer)
- Planification du voyage en amont
Pour l'instant, Itineria sert à **documenter** un voyage pendant/après qu'il se passe (étapes ajoutées au fil de l'eau). L'idée de la V3 est d'ajouter la possibilité d'**organiser le voyage avant le départ**, pour que l'appli couvre tout le cycle de vie du voyage : préparation -> vécu -> souvenir.
## Hors scope 
- Application mobile native
- Mode hors-ligne
- Import automatique depuis d'autres services (Google Photos, etc.)
